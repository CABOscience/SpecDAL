Introduction

SpecDAL is python library/toolkit for the the analysis of field spectroscopy data. It currently includes classes/functions for importing data from various file formats, processing the data, plotting the data and exporting the data into ascii files. The classes in SpecDAL can be easily configured (programatically) into custom processing pipelines.
